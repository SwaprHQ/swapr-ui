# Swapr UI Lib

This is the UI lib we use at Swapr to build web3 apps fast.

## Available Scripts

### Build the library

```
npm run rollup
```

### Publish the library

```
npm publish
```

### Run tests

```
npm run test
```

### Run storybook locally

```
npm run storybook
```
